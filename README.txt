How to run the tests:


1. Install chromedriver
brew update
brew install chromedriver

2. Install Safari driver
download:
https://github.com/SeleniumHQ/selenium/raw/master/javascript/safari-driver/prebuilt/SafariDriver.safariextz

3. Install firefox browser

4. Install jdk from http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

5. Start the Selenium server:
./node_modules/.bin/webdriver-manager start

6. grunt protractor-e2e
